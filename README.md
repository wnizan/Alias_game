# 🎮 אליאס - Alias Game

משחק ניחוש מילים חברתי בזמן אמת עם סנכרון בין מכשירים

## 🚀 התקנה מהירה

### דרך 1: Vercel (אונליין - בלי התקנה)
```
פשוט בקר בקישור: https://alias-game-seven.vercel.app
```

### דרך 2: לוקאלי (במחשב)
```bash
npm install
npm run dev
```

## 🎯 איך משחקים

### משחק לוקאלי:
1. בחר "🎮 משחק מהיר"
2. בחר מספר קבוצות
3. הזן טיימר
4. משחקים!

### משחק אונליין (עם Supabase):
1. בחר "👥 משחק אונליין"
2. בחרו "פתח משחק חדש" או "הצטרף"
3. שלחו קוד לחברים
4. משחקים בזמן אמת!

## 🔧 דרישות

- Node.js 16+ (לוקלי בלבד)
- Supabase Account (חינמי)
- API Keys (Project URL + Anon Key)
